Contract VersionÂ¶

The contract version denotes the data model and behavior that the API supports. The current contract version is v1. The requested contract version is included in all request URLs. Different contract versions of the API may be available at any given time and are not guaranteed to be compatible with one another.

Example Request URL (contract version v1)

https://rackspace.example.com/v1/1234

Note

This document pertains to contract version 1.
